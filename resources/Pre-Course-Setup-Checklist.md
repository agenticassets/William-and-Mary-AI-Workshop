## üö® Troubleshooting Quick Fixes

### Common Issues & Solutions

**n8n Login Problems**
- Clear browser cache and cookies
- Try incognito/private browsing mode
- Use different browser if needed

**API Key Issues**
- Verify API key is copied correctly (no extra spaces)
- Check usage limits on service provider dashboard
- Regenerate API key if needed

**Credential Connection Failures**
- Test API key on provider's website first
- Double-check service account permissions (for Google)
- Try re-creating credentials in n8n

**Workshop Project Access**
- Verify instructor email invitation was accepted
- Check project permissions in n8n settings ‚Üí Users
- Ensure you're in the correct project
